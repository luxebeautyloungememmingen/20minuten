ℹ️ Stempeluhr – Wozu dient diese Seite?

Die Stempeluhr ist das zentrale Werkzeug für die digitale Zeiterfassung im fitbox Studio. Jede Buchung wird direkt ins System gespeichert und ist die Basis für Lohnabrechnung, Überstundenberechnung und den Monatsabschluss. Kein Zettel, kein Nachtrag – alles läuft in Echtzeit.

So wird sie benutzt
1. Automatische Erkennung Die Seite erkennt dich automatisch anhand deines Logins. Dein Name und dein Studio werden direkt angezeigt – du musst nichts manuell auswählen.
2. Arbeitsbeginn → „Kommen" Sobald du deinen Dienst beginnst, drückst du Kommen. Die Uhrzeit wird sekundengenau gespeichert. Das ist der Startpunkt deiner Arbeitszeit.
3. Pause korrekt buchen Wenn du in die Pause gehst → Pause Start. Wenn du zurückkommst → Pause Ende. Pausenzeiten werden automatisch von deiner Arbeitszeit abgezogen.
4. Arbeitsende → „Gehen" Am Ende deines Dienstes drückst du Gehen. Damit ist der Arbeitstag abgeschlossen. Du kannst danach jederzeit einen neuen Dienst starten.
5. Notiz (optional) Das Notizfeld ist für besondere Hinweise – z. B. „Vertretung für Kollegin" oder „früher gegangen wegen Arzt". Nur ausfüllen wenn nötig.
Die 4 Buttons im Überblick
🟢 Kommen
Dienst beginnen – immer als erstes
🟡 Pause Start
Pausenbeginn erfassen
🔵 Pause Ende
Rückkehr aus der Pause buchen
🔴 Gehen
Dienst beenden – immer am Schluss
⚠️ Wichtige Regeln
  • Immer sofort stempeln – keine Buchungen nachträglich vornehmen.
  • Ausgegraute Buttons passen nicht zum aktuellen Status – das ist kein Fehler.
  • Vergessenes „Gehen" wird vom Admin nachgetragen und kann zu Rückfragen führen.
  • Deine Buchungen sind die Grundlage für Lohn & Überstunden – Genauigkeit zählt.
